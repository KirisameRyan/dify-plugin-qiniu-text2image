# 隐私政策 | Privacy Policy

**最后更新日期：2026-03-05**

## 1. 概述 / Overview
本插件（**七牛云文生图插件 / Qiniu Text to Image Plugin**）是一个 Dify Tool 插件，仅用于调用七牛云官方文生图 API（https://api.qnaigc.com/v1/images/generations）。

**本插件本身不收集、不存储、不处理任何用户个人数据。**

## 2. 数据处理说明 / Data Handling

### 用户输入的数据
- 提示词（prompt）
- 模型选择（model）
- 生成数量（n）
- 画面比例（aspect_ratio）
- 七牛云 API Key（仅本次调用使用）

**这些数据仅在本次调用时临时转发给七牛云官方 API，不会保存在插件或 Dify 服务器中。**

### API Key 处理
用户输入的七牛云 API Key **仅用于本次请求**，插件不会保存、记录或传输给任何第三方（除七牛云 API 本身）。

### 生成的图片
生成的图片以 base64 形式直接返回给 Dify 用户，插件不存储任何图片。

## 3. 第三方服务 / Third Party
本插件会将上述数据发送给**七牛云官方 API**。  
七牛云的隐私政策请参考：  
https://www.qiniu.com/agreements/privacy-right

## 4. 数据存储 / Data Storage
- 本插件**不存储**任何数据
- 不使用任何数据库
- 不记录日志中的敏感信息

## 5. 用户权利 / User Rights
您可以随时：
- 删除插件
- 撤销 API Key
- 停止使用本插件

## 6. 政策更新 / Policy Updates
本隐私政策可能会不定期更新，更新后将在 GitHub 仓库中发布。

# Privacy Policy

**Last updated: March 05, 2026**

## 1. Overview
This plugin (**Qiniu Text to Image Plugin**) is a Dify Tool plugin that only calls the official Qiniu Cloud Text-to-Image API.

**This plugin itself does not collect, store, or process any user personal data.**

## 2. Data Processing

### User Input
- Prompt
- Model selection
- Number of images (n)
- Aspect ratio
- Qiniu API Key (used only for the current request)

These data are **temporarily forwarded** to the Qiniu Cloud API for this single request and are **not stored** by the plugin or Dify.

### API Key
The API Key you provide is used **only for the current call** and is not saved or transmitted elsewhere.

### Generated Images
Images are returned directly to Dify as base64. The plugin does not store them.

## 3. Third Party Service
Data is sent to **Qiniu Cloud official API**.  
Please refer to Qiniu’s privacy policy:  
https://www.qiniu.com/agreements/privacy-right(https://www.qiniu.com/agreements/privacy-right)

## 4. Data Storage
- No database is used
- No user data is stored
- No logs contain sensitive information

## 5. Your Rights
You can delete the plugin or revoke your API Key at any time.

## 6. Updates
This privacy policy may be updated. The latest version will always be available in the GitHub repository.
